# AI Tooling Specialist

Welcome, and thank you for your interest.

This is designed to evaluate, ability to design practical AI automations, think through workflow architecture, and create a working prototype or scaffold.

## Project Brief

You have just joined a new startup building an AI-powered onboarding automation platform for enterprise clients. The company is pre-launch, has no AI tooling stack, and wants to automate and streamline the end-to-end new-hire onboarding experience.

### General Onboarding Process
1. Collect new hire information and documents
2. Set up accounts and access
3. Assign and track training and compliance modules
4. Schedule intro calls and orientation sessions
5. Provide a personalized onboarding plan and resources
6. Gather feedback and milestone check-ins

## Tasks

### Task 1 — AI-Powered Automation Solution (Design)
Design an AI-driven automation that improves and accelerates the onboarding workflow.

Your submission should clearly outline:
- Workflow logic
- Where and how AI is used
- Prompt engineering details, if any
- Data flows and integrations
- How the automation improves onboarding

### Task 2 — Implementation Demo (n8n or Zapier)
Create a working scaffold, prototype, or demo of your proposed automation.

This can include:
- n8n or Zapier workflow
- Code snippet
- Workflow diagram
- Screenshots
- Short video walkthrough

Your demo must show the core automation logic, even at a basic level.
